Requirements

A control node is any machine with Ansible installed. From the control node, you can run commands and playbooks from a laptop, desktop, or server. However, you cannot run IBM Power Systems HMC collection on a Windows system.

A managed node is often referred to as a target node, or host, and it is managed by Ansible. Ansible is not required on a managed node.

The nodes listed below require these specific versions of software:

Control node

  • Ansible version: 2.9 or later

  • Python: 3

Managed node

  • HMC level: V8R8.7.0 or V9R1 or later
